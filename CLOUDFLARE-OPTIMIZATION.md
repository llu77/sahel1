## 🛠️ **كيفية النشر**

### 1. **المتطلبات**
```bash
# تثبيت Wrangler CLI
npm install -g wrangler

# تسجيل الدخول
wrangler login
```

### 2. **التكوين**
قم بتحديث المعلومات في `wrangler.toml`:
```toml
account_id = "YOUR_ACCOUNT_ID"
zone_id = "YOUR_ZONE_ID"
```

### 3. **النشر**
```bash
# نشر سريع
bash deploy-cloudflare.sh

# أو يدوياً
wrangler deploy
wrangler pages deploy .next
```
